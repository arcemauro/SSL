## Enunciado
El objetivo de este trabajo es la preparación inicial del entorno de desarrollo y la publicación de un programa básico en C que imprima el mensaje "Hello, World!" por la salida estándar. Se debe demostrar la correcta configuración del compilador bajo el estándar C23 y la gestión de repositorios en GitHub.

## Hipótesis de Trabajo
* **Sistema Operativo:** [COMPLETAR: ej. Windows 11 / Ubuntu 22.04]
* **Compilador:** [COMPLETAR: ej. gcc 13.2.0 / clang 18.0]

## Resolución
1. Escribir el código en un archivo llamado `hello.c`.
2. Compilar el programa utilizando el estándar C23.
3. Ejecutar y redireccionar la salida a un archivo `output.txt`.
